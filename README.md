TypeScript + Vue + Vuetify

- なんでTypeScript？
- なんでVue？

業務案件をこのセットでやってて、それなりにいい感じに覚えたと思うので自分のプロダクトでも使おうと思ったからでした。

## Develop

Using Node.js v10.15.0

```bash
git clone git@github.com:oti/tkg-slot.git
cd tkg-slot
npm ci
```

### Local

```
npm run serve
```

### Build

```
npm run build
```
